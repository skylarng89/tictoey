# TicToey

A beautiful Tic-Tac-Toe game with smooth animations built with Flutter.

## Features

- **Smooth Gameplay**: Interactive 3x3 grid with responsive touch controls
- **Visual Feedback**: Confetti celebrations for wins and smooth animations
- **Haptic Feedback**: Vibration on moves for enhanced user experience
- **Score Tracking**: Keeps track of X wins, O wins, and draws
- **Modern UI**: Material 3 design with beautiful typography
- **Cross-Platform**: Works on Android, iOS, Web, and Desktop

## Getting Started

### Prerequisites

- Flutter SDK ^3.9.2
- Dart SDK (included with Flutter)

### Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```

## How to Play

1. Players take turns placing X and O on the 3x3 grid
2. First player to get 3 in a row (horizontal, vertical, or diagonal) wins
3. If all 9 squares are filled without a winner, it's a draw
4. Tap the reset button to start a new game

## Dependencies

- `google_fonts: ^6.1.0` - Beautiful typography
- `flutter_animate: ^4.2.0` - Smooth animations
- `confetti: ^0.7.0` - Celebration effects
- `vibration: ^1.8.6` - Haptic feedback
